# Metodología de las Ciencias Sociales

## Summary
Este módulo analiza la relación entre los métodos cuantitativos y cualitativos en la investigación social, y cómo seleccionar casos y variables para evitar sesgos y problemas de inferencia.

## Core ideas
- La investigación social puede ser cuantitativa o cualitativa
- La selección de casos y variables es crucial para evitar sesgos y problemas de inferencia
- La investigación social busca establecer relaciones causales entre variables
